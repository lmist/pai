## Edge Cases
- `FIXTURE_MODE=strict` + cache miss — must raise loudly, NOT silently fall through to OpenAI on stage.
- Malformed PDF upload — fall back to paste mode with clear error, do not crash.
- Criterion numeric boundary (A1c exactly 7.5% when threshold is ≥7.5%) — met, not unmet.
- `demo-reset` + `demo-seed` ordering — seed must complete before any UI renders; calibration graph with zero submissions is a critical silent failure.
- Rubric version mismatch (fixture cached against v1, code expects v2) — raise, don't silently score against wrong rubric.
- LLM structured-output schema violation — Pydantic catches, user sees "parsing failed" toast (should never fire in demo mode).
